### Como rodar
```

$ cd Spotper-Api
$ python3 -m venv venv (Criar Virtual Enviroment)
$ venv\Scripts\activate (IMPORTANTE)
$ pip install -r requirements.txt
(No visual studio)
Ctrl + p
Digite Python:Select Interpreter
Selecione o Virtual Enviroment Criado
```
